


# How to run:
```sh
# Setup
npm install
npm install -g typescript


# Running (2 separate terminals needed)
# Terminal 1:
npx webpack
# Terminal 2:
npx http-server -c-1 ./out
```

Visit http://localhost:8080 in your favoite browser!
